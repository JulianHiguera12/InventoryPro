# Contribuyendo a InventoryPro

Gracias por tu interés en contribuir a **InventoryPro**. Aquí hay algunas pautas para ayudarte a comenzar:

## Cómo Contribuir

1. **Haz un Fork del Repositorio**  
   Haz clic en el botón "Fork" en la parte superior derecha de la página del repositorio para crear tu propia copia del proyecto.

2. **Clona tu Fork**  
   Clona tu fork en tu máquina local:
   ```bash
   git clone https://github.com/TU_USUARIO/InventoryPro.git
   ```

3. **Crea una Nueva Rama**  
   Siempre crea una nueva rama para tus cambios:
   ```bash
   git checkout -b nombre-de-tu-rama
   ```

4. **Realiza los Cambios**  
   Haz los cambios que desees en el código.

5. **Commitea tus Cambios**  
   Asegúrate de que tus commits sean claros y concisos:
   ```bash
   git commit -m "Descripción de los cambios"
   ```

6. **Sube tus Cambios**  
   Sube tus cambios a tu fork:
   ```bash
   git push origin nombre-de-tu-rama
   ```

7. **Crea un Pull Request**  
   Ve a la página del repositorio original y haz clic en "New Pull Request". Describe tus cambios y crea el PR.

## Código de Conducta

Por favor, respeta el código de conducta del proyecto y asegúrate de interactuar de manera positiva y constructiva.

## Preguntas

Si tienes alguna pregunta, no dudes en abrir un problema o contactar a un mantenedor del proyecto.

Gracias por contribuir a **InventoryPro**. ¡Esperamos ver tus contribuciones!
