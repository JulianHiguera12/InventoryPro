# InventoryPro

## Project Overview
InventoryPro is a comprehensive inventory management system designed to streamline and automate processes for various types of businesses. It helps users keep track of inventory levels, manage stock movements, and generate insightful reports.

## Features
- User-friendly interface for easy navigation.
- Real-time inventory tracking.
- Stock alerts for low inventory.
- Detailed reporting and analytics.
- Multi-user support with role-based access.

## Technologies
- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT

## Installation
To get started with InventoryPro, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/JulianHiguera12/InventoryPro.git
   ```
2. Navigate to the project directory:
   ```bash
   cd InventoryPro
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Set up your environment variables:
   Create a `.env` file in the root of the project and add your configurations.
5. Start the development server:
   ```bash
   npm start
   ```

## Usage
After starting the server, visit `http://localhost:3000` in your browser to access the application. You can register for an account or log in if you already have one.
